# Course-Aware AI Teaching Assistant 

An open-source LLM-powered teaching assistant that enables the uploading of course materials and the formulation of questions based solely on the content supplied.

## Features
- Upload PDFs / text files
- Open-source Hugging Face models
- Retrieval-Augmented Generation (RAG)
- Streamlit UI

## Tech Stack
- Streamlit
- Hugging Face Transformers
- Sentence Transformers
- FAISS
- Python

## Educational Use
Perfect for:
- AI / ML courses

##  Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
